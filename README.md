# OBD2 MCP Server — "The Diagnostic Oracle"

A Model Context Protocol (MCP) server that gives GitHub Copilot (Agent Mode) expert-level automotive diagnostic knowledge. Highlight a DTC error code in your terminal or describe a symptom in chat and get a full mechanical breakdown — causes, symptoms, and fixes — instantly.

## Why This Exists

Modern AI coding assistants can do more than write code. With MCP, you can give them **domain-specific tools** they can call autonomously. This project turns Copilot into a mechanic's reference manual by exposing OBD-II diagnostic data through a local MCP server.

## MVP Scope

### Data Layer — `obd2.db`

A single SQLite database with one table:

**`dtc_codes`**

| Column        | Type | Example                          |
| ------------- | ---- | -------------------------------- |
| `code`        | TEXT | `P0300`                          |
| `description` | TEXT | `Random/Multiple Cylinder Misfire Detected` |
| `symptoms`    | TEXT | `Rough idle, shaking, flashing CEL` |
| `fix`         | TEXT | `Inspect and replace spark plugs, ignition coils, or fuel injectors` |

### Server Logic — `main.py`

Two tools exposed to Copilot via FastMCP:

- **`get_code_details(code: str)`** — Look up a specific DTC (e.g. `P0300`) and return its description, symptoms, and recommended fix.
- **`search_by_symptom(text: str)`** — Free-text search (e.g. *"my car is stalling"*) that returns matching DTC codes ranked by relevance.

### Copilot Connection — `.vscode/mcp.json`

A workspace-level config file that tells VS Code to start the MCP server when Copilot Chat opens:

```json
{
  "servers": {
    "obd2-diagnostic": {
      "type": "stdio",
      "command": "uv",
      "args": ["run", "main.py"]
    }
  }
}
```

## Getting Started

### Prerequisites

- Python 3.11+
- [uv](https://docs.astral.sh/uv/) (`curl -LsSf https://astral.sh/uv/install.sh | sh`)
- VS Code with GitHub Copilot extension (Agent Mode enabled)

### Installation

```bash
# Clone the repo
git clone https://github.com/<your-username>/obd2-mcp-server.git
cd obd2-mcp-server

# Install dependencies
uv sync

# Seed the database
uv run seed.py

# Test the server manually
uv run main.py
```

### Using with Copilot

1. Open the project in VS Code.
2. The `.vscode/mcp.json` config registers the server automatically.
3. Open Copilot Chat (Agent Mode) and try:
   - `What does code P0300 mean?`
   - `My car is shaking and the check engine light is flashing.`
   - `Look up P0420 for me.`
